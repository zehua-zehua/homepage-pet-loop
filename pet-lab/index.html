Homepage Evaluation Model

主页场景里，Loopi 不是单纯“可不可爱”的宠物评分，而是用 8 题问卷判断它作为个人品牌虚拟形象是否成立。每题 1-5 分：1 = 非常不同意，3 = 一般，5 = 非常同意。

总分 = 8 题平均分；维度 = 视觉第一印象、主页适配度、个人气质匹配、小马 + 小狗设定
A 视觉第一印象

判断 Loopi 是否好看，以及视觉质感是否适合一个专业个人主页。

B 主页适配度

判断 Loopi 放在 yuanzehua.me 是否自然，并且增强第一印象而不是分散注意力。

C 个人气质匹配

判断它是否传达好奇、AI 探索、亲和、容易交流的站主气质。

D 小马 + 小狗设定

判断它是否同时保留小马的行动力、成长感、向前感，以及小狗式亲和、快乐、陪伴感。
